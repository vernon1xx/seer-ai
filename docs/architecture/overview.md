## The 6-Face Cube

| Face | Position | Purpose |
|------|----------|---------|
| Facility Map | Top | Site navigation, building drill-down |
| Administration | Front | HR, policies, forms, directory |
| Safety | Right | LOTO, JSAs, training, incidents |
| Production | Back | Schedules, quality, inventory, KPIs |
| Maintenance | Left | CMMS, work orders, equipment |
| Environmental | Bottom | Air, water, waste, permits, regulations |

---

## Cube Configurator

The Cube Configurator allows administrators to customize the cube without coding:

- **Face Management**: Rename, reorder, theme faces
- **Tile Editor**: Add/edit/remove tiles, set icons and descriptions
- **Link Builder**: Configure links for each tile
- **Grid Options**: 2x2, 3x2, 3x3, 4x3 per face
- **Preview Mode**: Live 3D preview before publishing
- **Export/Import**: JSON configurations for templates

See [Cube Configuration Schema](cube-config-schema.md) for JSON format.
